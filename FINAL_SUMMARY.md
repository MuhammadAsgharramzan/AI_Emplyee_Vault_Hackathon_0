## 🚀 How to Use

### Quick Start (Bronze Tier)

```bash
# 1. Start the file system watcher
cd watchers
uv run python filesystem_watcher.py

# 2. In another terminal, test by dropping a file
cp test_file.md inbox/

# 3. Use Agent Skills
claude /update-dashboard
claude /process-inbox
claude /daily-briefing

# 4. View Dashboard
cat Dashboard.md
# Or open in Obsidian for better viewing
```

### Advanced Usage (Silver Tier)

#### Using the Orchestrator
```bash
# Start all watchers
./orchestrator.sh start

# Check status
./orchestrator.sh status

# Stop all watchers
./orchestrator.sh stop

# Generate daily briefing
./orchestrator.sh daily-briefing
```

#### Gmail Monitoring
```bash
# First time setup (requires Google Cloud credentials)
cd watchers
uv run python gmail_watcher.py
# Browser will open for authentication
# Token saved for future runs
```

#### WhatsApp Monitoring
```bash
# First time setup (requires QR code scan)
cd watchers
uv run python whatsapp_watcher.py
# Browser will open - scan QR code with phone
# Session saved for future runs
```

#### LinkedIn Posting
```bash
# Generate post
claude /linkedin-post

# Review and approve
# Edit need_action/linkedin_posts/LINKEDIN_POST_*.md
# Change status to "approved"

# Post to LinkedIn
cd watchers
uv run python linkedin_automation.py
```

#### Email Operations
```bash
# Send email
cd watchers
uv run python email_mcp_server.py send "to@example.com" "Subject" "Body"

# Create draft
uv run python email_mcp_server.py draft "to@example.com" "Subject" "Body"
```
