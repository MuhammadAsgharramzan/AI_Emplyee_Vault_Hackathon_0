# Generate Daily Briefing

Create a daily briefing summarizing the AI Employee's activities and current status.

## Instructions

You are generating a daily briefing for the human supervisor.

1. **Review all relevant files:**
   - Dashboard.md for current stats
   - Company_Handbook.md for context
   - Done folder for completed tasks
   - need_action folder for pending items

2. **Create a briefing** that includes:
   - **Executive Summary**: 2-3 sentence overview
   - **Completed Today**: List of tasks finished
   - **Pending Items**: What's waiting for action
   - **Alerts**: Any issues or concerns
   - **Recommendations**: Proactive suggestions
   - **Next Steps**: What will happen next

3. **Save the briefing** to `Briefings/YYYY-MM-DD_Daily_Briefing.md`

4. **Update the Dashboard** to reflect the briefing was generated

## Briefing Template

```markdown
---
generated: [ISO timestamp]
period: [date]
---

# Daily Briefing - [Date]

## Executive Summary
[2-3 sentences summarizing the day]

## Completed Tasks
- [Task 1]
- [Task 2]

## Pending Items
- [Item 1] - Priority: [high/medium/low]
- [Item 2] - Priority: [high/medium/low]

## Alerts
- [Any issues or concerns]

## Recommendations
- [Proactive suggestions based on patterns]

## Next Steps
- [What will happen next]

---
*Generated by AI Employee*
```

## Output

Confirm the briefing was created and provide the executive summary.
