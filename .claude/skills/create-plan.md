# Create Task Plan

Create a detailed plan for multi-step tasks that require coordination.

## Instructions

You are creating a structured plan for a complex task. Follow these steps:

1. **Understand the task:**
   - Read the task description carefully
   - Identify all requirements and constraints
   - Check Company_Handbook.md for relevant rules

2. **Break down the task** into clear steps:
   - Each step should be actionable
   - Identify dependencies between steps
   - Estimate complexity (simple/medium/complex)
   - Flag steps requiring human approval

3. **Create a plan file** in the format below

4. **Save to need_action folder** as `PLAN_[task-name]_[date].md`

5. **Update Dashboard** to reflect new plan created

## Plan Template

```markdown
---
created: [ISO timestamp]
task: [task name]
status: pending
priority: [high/medium/low]
requires_approval: [yes/no]
---

# Task Plan: [Task Name]

## Objective
[Clear statement of what needs to be accomplished]

## Context
[Background information and why this task is needed]

## Steps

### Step 1: [Step Name]
- **Action:** [What needs to be done]
- **Complexity:** [simple/medium/complex]
- **Dependencies:** [None or list other steps]
- **Approval Required:** [yes/no]
- **Status:** [ ] Pending

### Step 2: [Step Name]
- **Action:** [What needs to be done]
- **Complexity:** [simple/medium/complex]
- **Dependencies:** [Step 1]
- **Approval Required:** [yes/no]
- **Status:** [ ] Pending

[Continue for all steps...]

## Success Criteria
- [ ] [Criterion 1]
- [ ] [Criterion 2]

## Risks & Considerations
- [Risk 1 and mitigation]
- [Risk 2 and mitigation]

## Resources Needed
- [Resource 1]
- [Resource 2]

## Notes
[Any additional context or observations]

---
*Plan created by AI Employee*
```

## Output

Confirm the plan was created and provide a summary of the steps.
