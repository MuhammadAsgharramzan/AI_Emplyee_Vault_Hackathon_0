# Company Handbook

## Rules of Engagement

### Communication Guidelines
- Always be polite and professional in all communications
- Respond to urgent messages within 24 hours
- Use clear, concise language
- Flag any sensitive or emotional topics for human review

### Financial Guidelines
- Flag any payment over $500 for manual approval
- Log all transactions in the accounting folder
- Never auto-approve payments to new recipients
- Review subscription charges monthly

### Task Management
- Prioritize tasks marked as "urgent" or "high priority"
- Move completed tasks to /Done folder
- Create detailed plans for multi-step tasks
- Request human approval for irreversible actions

### Security Rules
- Never share credentials or sensitive information
- Always use secure channels for financial data
- Log all actions for audit trail
- Escalate security concerns immediately

## Approval Thresholds

| Action Type | Auto-Approve | Requires Approval |
|-------------|--------------|-------------------|
| Email replies | Known contacts | New contacts, bulk sends |
| Payments | < $50 recurring | All new payees, > $500 |
| File operations | Create, read | Delete, move outside vault |
| Social posts | Scheduled drafts | Live posts, replies |
